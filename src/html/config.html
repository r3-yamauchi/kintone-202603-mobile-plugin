参考: kintone 2026年3月アップデート情報

使い方

  1. プラグインを適用するアプリを選択し、このプラグインを追加します
  2. モバイル端末/モバイル表示モード、またはPCブラウザでkintoneアプリを開きます
  3. レコード一覧画面の上部に5つのテストボタンが表示されます（モバイル/PCそれぞれで対応APIを実行）
  4. 各ボタンをタップして、APIの動作を確認できます

機能

  • ボトムシート表示（kintone.mobile.createBottomSheet）
  • 確認ボトムシート表示（キャンセルあり/なし, kintone.mobile.showConfirmBottomSheet）
  • 通知バナー表示（SUCCESS/INFO/ERROR, kintone.mobile.showNotification）
  • ローディング表示・消去（kintone.mobile.showLoading）
