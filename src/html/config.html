機能

  • モバイル版レコード一覧画面に4つのテストボタンを表示
  • ボトムシート表示機能のテスト（kintone.mobile.createBottomSheet）
  • 確認ボトムシート表示機能のテスト（kintone.mobile.showConfirmBottomSheet）
  • 通知メッセージ表示機能のテスト（kintone.mobile.showNotification）
  • ローディング表示機能のテスト（kintone.mobile.showLoading）

使い方

  1. プラグインを適用するアプリを選択し、このプラグインを追加します
  2. モバイル端末またはモバイル表示モードでkintoneアプリを開きます
  3. レコード一覧画面の上部に4つのテストボタンが表示されます
  4. 各ボタンをタップして、新しいモバイルAPIの動作を確認できます
注意: このプラグインはモバイル版専用です。PC版では動作しません。
参考: モバイルAPI更新情報（2026年3月）
